# What is SGBD?

> SGBD files (often with .PRG or .GRP extensions) are ECU Description Files > (Steuergeräte Beschreibungsdatei) used by BMW’s EDIABAS diagnostic software to communicate with vehicle modules. They contain protocols, diagnostic trouble codes, and parameters required for tools like INPA and WinKFP to read, flash, or diagnose Electronic Control Units (ECUs).
